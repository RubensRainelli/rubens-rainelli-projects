## 🛠 Note Tecniche e Competenze Dimostrate

La struttura di questa SPA dimostra un approccio ingegneristico orientato alla massimizzazione delle performance e alla riduzione delle dipendenze esterne:
- **Vanilla JavaScript**: Gestione dello stato e del rendering dinamico dei dati (da file JSON) senza l'uso di framework pesanti.
- **CSS3 Moderno**: Utilizzo di custom properties (variabili CSS), effetti di animazione avanzati (glitch, scanlines), design completamente responsive e compatibilità cross-browser (compresi i fallback per le proprietà di scrollbar standard e legacy).
- **Internazionalizzazione (i18n)**: Rilevamento dinamico della lingua del browser con persistenza locale (`localStorage`) ed overlay interattivo per la scelta della lingua e accettazione EULA.
